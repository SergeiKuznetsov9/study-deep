В этом уроке организуем стили приложения таким образом, чтобы можно было
добавлять различные темы, добавлть глобальные шрифты и т.д.

Создадим папку стилей и поработаем с ней. Закинем туда файл с глобальными стилями,
создадим файл, обнуляющий стили, файлы стилей тем и переменных.

Все файлы стилей импортируем в главный файл стилей, который в свою очередь будет
импортирован в файл входа

Начнем с добавления переменных.

Затем подключим шрифт в корневом файле стилей

Затем работаем с темами и тоже подключаем их

Для того, чтобы тему можно было менять из различных частей программы,
создадим контекст.

Также создадим провайдер для использования контекста. По своей сути - это будет
компонент, который вернет из контекста провайдер. Также закинем туда код для
смены темы.

Также логику по получениб темы и ее смены следует вынести в отдельный хук
