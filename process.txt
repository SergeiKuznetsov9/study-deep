Начинаем с npm init, а также создаем всякие дирректории

Выпонение файла из командной строки node src/index.js

Импорт/экспорт по дефолту работать не будет. Это можно исправить используя вебпак. Для его
использования установим нужные пакеты:
npm i -D webpack@5.69.1 webpack-cli@4.9.2

Далее создаем файл webpack.config.js и пишем в нем код (см файл)

Для запуска сборки в терминале достаточно написать npm run build, предварительно добавив 
скрипт в package.json

После настройки создаем index.html, в этом файле будет наша верстка. После его создания
Вебпаку нужно сообщить где он лежит. Кроме того в него нужно будет встраивать скрпты, которые
мы будем писать. Это также может делать вебпак при помощи различных плагинов

npm install -D html-webpack-plugin@5.5.0

Далее добавляем TypeScript. Для этого нужно добавить сам тайпскрипт и его лоудер
npm i -D typescript@4.5.5 ts-loader@9.2.6

После этого нужно создать ts.config.json и поменять расширение фалйов на ts
