В этом уроке создадим виджет переключения темы

Для виджета добавим иконки SVG и соответственно лоудер для них

Однако проект все равно не увидит svg файлы и причина этого - ТайпСкрипт не
видит эти файлы, ему нужно рассказать о них. И для этого в глобайльный файл типов 
добавим такую запись:

declare module "*.svg" {
  const content: React.FunctionComponent<React.SVGAttributes<SVGElement>>;
  export default content;
}

после этого все работает. SVG лоудер преобразовывает иконки в обычные реакт-ком-
поненты и у них теперь есть все соответсвующие пропсы

Также сразу добавим файл-лоудер, который позволяет работать с другими расширениями
(картинками, шрифтами). Но нужно указывать регулярку!!!

Также добавим свою кастомную кнопку по аналогии с AppLink
В данной связи следует обратить внимание на то, каким образом создается интерфейс
пропсов:

interface ButtonProps extends ButtonHTMLAttributes<HTMLButtonElement> {
  className?: string;
}

Т.е. у нас будут все стандартные атрибуты в пропсах и те, которые добавим мы
