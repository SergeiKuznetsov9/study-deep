В этом уроке сделаем имитацию бэкенда

Установим json server
npm install -D json-server

Далее в отдельной папке создадим БД и наполним ее примером из гитхаба

Команда для запуска сервера - json-server --watch ./json-server/db.json --port 8000

Далее сделаем простой сервер с имитацией проверки токена

Добавим скрипт в package.json, который будет запускать файл с кодом поднятия сервера

Теперь, запустив, сервер, мы сможем получить данные только в том случае, если в
хедере будет какой-нибудь токен (проверяется сам факт наличия, а не конкретная строка)
