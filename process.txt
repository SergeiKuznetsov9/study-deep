В этом уроке добавим тестовую среду Jest и настроим ее

Установим пакет
npm install --save-dev jest

Далее сгенерируем базовый конфигурационный фалй для jest:
npm init jest@latest.

Поотвечаем на вопросы и перенесем сгенерированный файл в папке jest в config.

В данном файле произведем нужные настройки.

Также допишем добавленный скрипт, поскольку файл конфига мы перенесли из корня

Для написания тестов на TS также придется установить типы:
npm install --save-dev @types/jest

Далее приступим к написанию тестов. Напишем первый тест и тест не пройдет, т.к.
